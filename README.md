# shor

Shor's factoring algorithm written in rust

```
$ cargo install shor
```

```
$ shor 15
N: 15 (a: 8, t: 3)
0100001 +0.2500 +0.0000 0.0625 s/r= 1/ 4; p=3, q=5
1100001 +0.2500 +0.0000 0.0625 s/r= 3/ 4; p=3, q=5
```
